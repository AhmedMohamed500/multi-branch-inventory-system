# Frontend Architecture

- `app/`: مسارات Next.js وواجهتا Login وApplication.
- `components/`: الـApp Shell والصفحات والجداول والنوافذ التفاعلية.
- `data/`: بيانات Mock فقط، دون استدعاء مباشر من الصفحات.
- `services/`: طبقة Promise تحاكي API وتفصل الواجهة عن مصدر البيانات.
- `types/`: العقود المشتركة الجاهزة للـBackend.
- `lib/`: منطق الحساب والتحقق القابل للاختبار.
- `messages/`: نصوص العربية والإنجليزية.

تتم إدارة حالة العرض محليًا في React، وتُحفظ تفضيلات اللغة والثيم والجلسة التجريبية في Local Storage.
